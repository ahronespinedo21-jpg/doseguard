## 🎬 Animation Effects

### **1. Page Load**
```
Initial:     Faded out, shifted down 30px
↓
0.8s later:  Fades in and slides up smoothly
            (cubic-bezier easing for bounce effect)
```

### **2. Form Elements**
```
Email Input:     Appears at 0.15s
Password Input:  Appears at 0.30s (staggered)
Button:          Appears at 0.45s (last)
```

### **3. Logo**
```
Continuously floats up/down:
  ↑ (up 10px)
  ↓ (down 10px)
  Duration: 3 seconds per cycle
```

### **4. Background Orbs**
```
Three colored circles:
  • Purple (top-left): scales 1.0 → 1.1 → 0.9
  • Blue (top-right):  same, but delayed 2 seconds
  • Pink (bottom):     same, but delayed 4 seconds
```

### **5. Gradient Shift**
```
Background gradient position:
  Smoothly moves left → right → left
  Duration: 15 seconds (continuous loop)
```
